# 🌿 AgriVision AI

**AgriVision AI** is a full-stack, AI-powered smart agriculture platform designed to assist farmers in the early detection and management of tomato leaf diseases. Using image analysis and machine learning, the system identifies plant diseases from uploaded leaf images and provides treatment recommendations, confidence scores, pesticide suggestions, fertilizer guidance, and downloadable PDF reports.

The platform features a modern and responsive interface built with **Next.js**, a high-performance **FastAPI** backend, secure JWT authentication, role-based access control, and a scalable database architecture. It supports multiple user roles, including **Farmers**, **Agricultural Experts**, and **Administrators**, making it suitable for real-world agricultural monitoring and research.

## ✨ Key Features

* 🌱 AI-powered tomato leaf disease detection
* 📸 Instant image upload and disease analysis
* 🎯 Confidence score with detailed prediction reports
* 💊 Personalized pesticide and fertilizer recommendations
* 📄 Downloadable PDF diagnosis reports
* 📊 Prediction history and analytics dashboard
* 👨‍🌾 Farmer dashboard for crop monitoring
* 👥 Admin panel for user and disease management
* 🔐 Secure JWT authentication with role-based authorization
* ⚡ RESTful API built using FastAPI
* 📱 Responsive UI built with Next.js and Tailwind CSS
