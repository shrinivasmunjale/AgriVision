================================================================================
                           UPLOAD FLOW
================================================================================

1. **Upload Images:**
   - POST /api/v1/predictions/upload
   - Files saved to backend/uploads/
   - Returns: ["/uploads/uuid1.jpg", "/uploads/uuid2.jpg"]

2. **Analyze Images:**
   - POST /api/v1/predictions/analyze
   - Calls ml_service.predict_disease()
   - Returns random disease + confidence

3. **Generate Recommendations:**
   - Queries database for pesticides/fertilizers
   - Uses text similarity matching
   - Returns top 3 recommendations

4. **Create Prediction Record:**
   - Saves to database
   - Links to user, disease, recommendations
   - Available in history

5. **Generate PDF Report:**
   - GET /api/v1/predictions/{id}/download
   - Creates PDF with ReportLab
   - Includes image, disease, recommendations
