================================================================================
                          API ENDPOINTS
================================================================================

All endpoints: http://localhost:8000/api/v1/...

PUBLIC:
  POST /auth/register       - Create account
  POST /auth/login          - Get JWT token
  GET  /auth/me             - Get current user

AUTHENTICATED:
  POST /predictions/upload  - Upload images
  POST /predictions/analyze - Analyze images
  GET  /predictions         - List predictions
  GET  /predictions/{id}    - Get prediction details
  GET  /predictions/{id}/download - Download PDF

ADMIN ONLY:
  GET    /admin/users       - List users
  POST   /admin/users       - Create user
  PUT    /admin/users/{id}  - Update user
  DELETE /admin/users/{id}  - Delete user
  GET    /admin/diseases    - List diseases
  POST   /admin/diseases    - Create disease
  GET    /admin/analytics   - System stats

================================================================================
                          TROUBLESHOOTING
================================================================================

✅ All major issues resolved!

If issues occur:

**Login fails:**
- Clear browser cache
- Check network tab (F12)
- Verify: http://localhost:8000/api/v1/auth/login

**Images don't show:**
- Already fixed! URLs now include full backend URL
- Check: http://localhost:8000/uploads/{filename}

**Upload fails:**
- Check file size (< 15MB)
- Check file type (images only)
- View backend terminal for errors

**404 errors:**
- Ensure backend is running on port 8000
- Ensure frontend is running on port 3001
- Check .env.local has correct API_URL
