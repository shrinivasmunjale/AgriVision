================================================================================
                          HOW IT WORKS
================================================================================

Current System:
1. Farmer captures images with drone
2. Downloads images from drone
3. Uploads to AgriVision AI
4. System analyzes each image
5. Identifies diseases in aerial shots
6. Provides treatment recommendations
7. Can process entire field coverage

Supported:
✓ DJI drones (Phantom, Mavic, Agras)
✓ Parrot drones (Bluegrass, Sequoia)
✓ Any agricultural drone
✓ Any camera (RGB, multispectral)
✓ Any image format (JPG, PNG, TIFF)
